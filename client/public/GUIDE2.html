🚀 Getting Started

System Requirements

  • Web Browser: Chrome, Firefox, Safari, or Edge
  • Internet Connection: Required for bank access
  • JavaScript: Must be enabled
  • Google Account: Optional but recommended

First-Time Setup

  1. Open the application (http://localhost:3000)
  2. Choose your language (English or עברית)
  3. Fill in bank credentials
  4. Start your first scrape!

🚀 כיצד להתחיל

דרישות מערכת

  • דפדפן: Chrome, Firefox, Safari, או Edge
  • חיבור לאינטרנט: נדרש לגישה לחשבון
  • JavaScript: חייב להיות מופעל
  • חשבון Google: אופציונלי אך מומלץ

  3. הכנס את פרטי הבנק שלך

Why Use Google Sheets?

Setup Steps

🔑 API Keys & OAuth

Understanding the Credentials

  • OAuth 2.0: Secure user authentication
  • Client ID: Public application identifier
  • Client Secret: Private key (keep it secret!)
  • Master Key: Local encryption password

Master Key Best Practices

  • Use 12+ characters with mixed case and symbols
  • Store in a password manager
  • Never share with anyone
  • If forgotten, profiles cannot be recovered

🔑 מפתחות API ו-OAuth

הבנת הפרטים

  • OAuth 2.0: אימות משתמש מאובטח
  • Client ID: מזהה אפליקציה ציבורי
  • Client Secret: מפתח פרטי (שמור בסוד!)
  • Master Key: סיסמת הצפנה מקומית

שיטות טובות ל-Master Key

  • השתמש ב-12+ תווים עם אותיות גדולות וסמלים
  • אחסן במנהל סיסמאות
  • לא תשתף עם אף אחד
  • אם שכחת, לא ניתן להחזיר פרופילים

🔄 Running Your First Scrape

Step-by-Step

  1. Select Provider: Choose your bank/credit card
  2. Enter Credentials: Username and password
  3. Configure Options: Date range, timeout, etc.
  4. Optional - Save Profile: For quick access next time
  5. Click Start Scrape: Wait for results
  6. View Results: In Results Explorer tab

💾 Working with Profiles

What Are Profiles?

Profiles securely store your bank credentials encrypted with your Master Key.

Creating a Profile

  1. Fill in your bank credentials
  2. Check "Save Profile" box
  3. Give it a name (e.g., "My Bank Account")
  4. Enter your Master Key
  5. Click Save

Using a Profile

  1. Click "Start from Profile" dropdown
  2. Select your saved profile
  3. Credentials auto-fill
  4. Click Start Scrape

💾 עבודה עם פרופילים

מה זה פרופיל?

פרופילים מאחסנים בצורה מאובטחת את פרטי הבנק שלך מוצפנים עם Master Key שלך.

יצירת פרופיל

  1. הזן את פרטי הבנק שלך
  2. סמן את תיבת "שמור פרופיל"
  3. תן לו שם (למשל "חשבון הבנק שלי")
  4. הזן את ה-Master Key שלך
  5. לחץ שמור

שימוש בפרופיל

  1. לחץ על רשימת "התחל מפרופיל"
  2. בחר את הפרופיל השמור שלך
  3. הפרטים מתמלאים באופן אוטומטי
  4. לחץ על התחל Scrape

⏱️ Automation & Scheduling

Scheduler Setup

  1. Click "Automation" tab
  2. Enable Scheduler
  3. Set time to run (e.g., 8 AM daily)
  4. Select profiles to scrape
  5. Enable "Save to Sheets"
  6. Save settings

Common Schedules

  • Daily at 8 AM - get fresh data every morning
  • Weekly on Monday - weekly financial review
  • Monthly on 1st - monthly summary

⏱️ אוטומציה וזמנוי

הגדרת מתכנן

  1. לחץ על כרטיסייה "אוטומציה"
  2. הפעל מתכנן
  3. קבע זמן הפעלה (למשל 8 בבוקר יומי)
  4. בחר פרופילים ל-scrape
  5. הפעל "שמור ל-Sheets"
  6. שמור הגדרות

לוחות זמנים נפוצים

  • יומי בשעה 8 בבוקר - קבל נתונים טריים כל בוקר
  • שבועי בשני - סקירה פיננסית שבועית
  • חודשי ב-1 - סיכום חודשי

🛠️ Troubleshooting

Common Issues

  • Credentials Rejected: Check username/password directly on bank website
  • Timeout Error: Increase timeout or try at different time
  • Cannot Decrypt Profile: Wrong Master Key - check password manager
  • Google Sheets Failed: Re-authorize or check API enablement

Getting Help

  • Check the Logs tab for error messages
  • Try clearing browser cache
  • Use incognito/private mode
  • Check GitHub issues page

🛠️ פתרון בעיות

בעיות נפוצות

  • פרטים דחויים: בדוק שם משתמש/סיסמה ישירות באתר הבנק
  • שגיאת timeout: הגדל timeout או נסה בזמן אחר
  • לא יכול לפענח פרופיל: Master Key שגוי - בדוק מנהל סיסמאות
  • Google Sheets נכשל: אשר מחדש או בדוק הפעלת API

קבלת עזרה

  • בדוק את כרטיסייה Logs לקבלת הודעות שגיאה
  • נסה לנקות את זיכרון מטמון של הדפדפן
  • השתמש במצב incognito/פרטי
  • בדוק דף GitHub issues
